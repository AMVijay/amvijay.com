# Javascript - async, await, promise understanding

## Introduction
async, await and promise are the important features in Javascript programming language. Here, will explain what this feature is and how it can be implemented. 

Javascript file will be executed in Javascript engines, mostly it will be browser, nodejs. Javascript engines architecture is Event Loop based, single thread, non I/O blocking.

### async
As name defines, this keyword used in function definition. This keyword tells javascript engine to make the defined function execution asynchronously. Typically javascript executes the inbuilt functions setTimeout, fetch asynchrnously. But user defined functions would expect async keyword to make the execution asynchronously.
